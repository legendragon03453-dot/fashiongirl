# **App Name**: FashionGirl Blog

## Core Features:

- Responsive Homepage Layout: Display sections for featured articles, content categories, product catalog, company history, and contact information, adapted for different screen sizes.
- Interactive Hero Carousel: A full-width image carousel on the homepage with automatic slide transitions, manual navigation buttons, slide indicators, and engaging titles.
- Blog Post Listings: Present a grid of blog posts, displaying an image, title, category, and author, allowing users to browse and click through to detailed articles.
- Product Showcase Carousel: A horizontally scrollable section showcasing various fashion items from the catalog, including images and product names.
- Dynamic Page Navigation: Implement clear navigation links in the header, supported by a mobile-friendly hamburger menu that slides in and out.
- Scroll-Animated Content: Introduce visually appealing fade-in-up animations for key content sections as users scroll down the page, enhancing engagement.
- AI-Generated Blog Post Titles (tool): Assist content creators by suggesting creative and SEO-friendly titles for new blog posts based on their drafts or keywords.

## Style Guidelines:

- Primary color: A sophisticated, dark charcoal (#18191A) to provide a strong foundation for text and interactive elements, evoking elegance and boldness.
- Background color: A very light, cool grey (#FAFBFC) for the main background, offering a clean, airy feel that complements the dark primary color.
- Accent color: A muted lavender-grey (#A18CB6) selected for subtle highlights and complementary elements, providing a soft contrast that aligns with a refined fashion aesthetic.
- Headlines (H1, H2): 'Quandco' (serif), loaded via custom font URL, to convey a unique and distinctive brand personality. Body text: 'Inter' (sans-serif) for legibility and a modern, neutral feel across all content. Note: currently only Google Fonts are supported for seamless integration with Next.js.
- Use minimalist, clean SVG icons for navigation (e.g., hamburger menu, carousel arrows) to maintain a sleek and modern visual style.
- Utilize a responsive grid and flexbox system, constrained by a max-width container, ensuring content is well-aligned and adapts fluidly to all screen sizes, from mobile to desktop.
- Implement smooth transition animations for hero carousel image zooms and element hovers, combined with elegant 'fade-in-up' animations for content sections upon scroll, creating a dynamic user experience.
